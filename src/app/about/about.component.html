Welcome To Zippy

Zippy is an opinionated starter template that uses: Angular + TypeScript and Tailwind CSS.

The starter currently has these following sample pages:

  • A dashboard 'Home' with Header and Footer.
  • A 'Sign In' page that is not hooked up to auth (when you click sign in, it checks a hard-coded username & password), on successful authentication it will redirect you to home page.
  • An 'About' page that describes the current features of this template.
  • This project is created using:
    • Angular CLI: 15.0.3
    • Node: 18.12.1
    • Package Manager: npm 9.2.0
    • OS: win32 x64
